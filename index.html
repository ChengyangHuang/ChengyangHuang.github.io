Projects

Domain Generalization via Ensemble Method for Microbes

This project aims to develop an ensemble method for the domain generalization problem and apply it to predict interactions between nanoparticles and microbes.

Generative Adversarial Networks (GANs) for Parameter Estimation

This is a proof-of-concept project of developing a generative model using GANs to estimate the parameter distribution from experimental measurements, which can be further applied to various engineering problems (e.g. mechanical, aerospace, biomedical). Current results show that this method is about 10 times faster than MCMC with similar performance in a test problem.
